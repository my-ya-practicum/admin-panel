# Init development

1) init poetry
```bash
$ poetry install
```

2) env
```
$ cp ./.env.template ./.env.local
$ cp ./.env.template ./.docker.env
```

`POSTGRES_USER=DB_USER=` - пользователь БД
`POSTGRES_PASSWORD=DB_PASSWORD=` - пароль пользователя
